Deixe a opção "Trocar Autor" dos produtos WooCommerce sempre disponível com esse leve plugin, sem acessos indevidos ou riscos para seu site.

== Description ==

Este plugin possibilita trocar o autor de um produto com facilidade, 

mantendo a segurança do seu site sem acessos indevidos 

e leve para não atrapalhar o desempenho de seu website.
